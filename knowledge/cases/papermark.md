# Papermark — 开源 DocSend 替代，1 年从 $1k 到 $45k MRR

## Timeline
- 2023-08：项目在 GitHub 开源
- 2023-11：$1k MRR（约 3 个月后首个付费）
- 2024-01：$5k MRR
- 2024-06：$20k MRR
- 2024-10：$45k MRR
- 2025-04：$500k ARR 里程碑

## 做对的事
- **开源作为信任杠杆**：GitHub star 增长即是营销，零广告成本获客
- **垂直替代 incumbent**：直接对标 DocSend 这一特定痛点（投资者 deck 分享 + tracking）
- **B2B SMB 定价**：$39-199/月 分层，比企业市场低但比 C 端高
- **3 个月内首单**：说明定位精准，不需要漫长冷启动
- **创始人透明**：IndieHackers 上公开 MRR 进度，反向滋养社区信任

## 做错的事
- 早期独自一人运维 7×24 客服压力大，后期才招第一个员工
- 初期功能过度扩展（追求 feature parity with DocSend），后来回退到核心功能

## 关键引语
> "Open source was our best marketing channel. Every GitHub star was a future customer."

## 评分启示
- **收入天花板维度**：开源 SaaS 的 P95 arc 是 12 个月 $1k → $45k MRR，注意这是 P95 不是中位数
- **冷启动维度**：垂直替代 incumbent 的策略可以把冷启动压缩到 3 个月首单（快于 P50 的 6 个月）
- **可持续性维度**：开源带来的 SEO + 社区信任是复利资产
- **技能匹配维度启示**：该案例验证了"后端程序员 + SaaS 架构"直接变现路径
- **注意**：Papermark 创始人有 YC 背景 + 欧洲市场地缘优势，非完全白板；复制难度高于表面看起来
