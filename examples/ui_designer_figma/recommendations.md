# 副业推荐报告

> 基于画像：UI 设计师 / 10h 周末 / 0 预算 / 不露脸不直播
> 生成时间：2026-04-16

以下方向基于 2026 年 4 月的公开信息交叉验证，按总分降序排列。

---

## #1 Figma 模板 + 小红书封面模板售卖（Gumroad + 小红书）  26/30 分

评分明细：
- 技能匹配 5 — Figma 是本职工具，封面/UI Kit 模板几乎不需要额外学习
- 时间适配 4 — 10h/周足够每周产出 1-2 套模板，制作可分块
- 启动成本 5 — 0 元，Figma Community 发布 + Gumroad 开店均免费
- 收入天花板 4 — Gumroad 公开案例（内容创作者 1.5 年累计 $7000+）；海外抽成后到手可观
- 冷启动 3 — 需要先做 3-5 套样品 + 小红书笔记引流，首单约 4-8 周
- 可持续性 5 — 数字产品典型被动收入，产品越多长尾越稳

匹配理由：直接用现有 Figma 技能，0 成本，不露脸符合排除项，双市场（海外 Gumroad + 国内小红书）对冲平台风险。
风险提示：小红书虚拟资料 2026 新规要求粉丝 ≥1000、月流水 ≥6000 才能开虚拟店，国内端起步前需积累粉丝。
冷启动第一步：本周在 Figma Community 发 1 套免费"小红书封面模板"引流，标注个人小红书账号。

## #2 插画约稿（闲鱼挂板 + 小红书引流）  24/30 分

评分明细：
- 技能匹配 5 — 插画 + iPad/Pencil 已就绪，起号即能接单
- 时间适配 4 — 10h 周末可稳定接 2-3 单，但修改沟通会占用工作日碎片
- 启动成本 5 — 0 元，设备齐备
- 收入天花板 3 — 新手单价 50-300 元，熟手 500-1500/单，月入 2000-5000 合理
- 冷启动 4 — 先做 3 个不同风格样品挂闲鱼，2-3 周内出单属正常
- 可持续性 3 — 按件计费，客户需要反复积累；有好评沉淀但复利弱

匹配理由：插画技能变现路径最短；线上沟通无需出镜，社恐友好。
风险提示：新手易被"先发货后付款"、脱离平台交易的单骗稿；务必收 30-50% 定金，走平台担保。
冷启动第一步：用周末 4 小时画 3 幅差异化样稿（赛璐璐 / 治愈系 / 线稿），周日挂闲鱼，标价 80-200。

## #3 UI 设计接单（闲鱼 + 猪八戒 / ZCOOL）  23/30 分

评分明细：
- 技能匹配 5 — 本职技能直接复用
- 时间适配 3 — 接单周期常不可控，10h/周偶尔会紧张
- 启动成本 5 — 0 元
- 收入天花板 4 — 单价 500-3000/单，熟手月入 3000-8000
- 冷启动 4 — 平台成熟，有作品集即可 2-3 周出单
- 可持续性 2 — 纯时间换钱，价格战严重，主业忙时副业就停

匹配理由：本职直接变现，单价最高的一档。
风险提示：这条路容易变成"第二份工作"；如果你想要的是喘息空间而非更多收入，谨慎选这条。
冷启动第一步：整理 3 个现有作品（脱敏），在闲鱼发"UI 设计接单"商品，标价按小时 150-300。

## #4 设计类图文自媒体（小红书 Figma 教程 + 作品）  22/30 分

评分明细：
- 技能匹配 4 — 需要补选题 + 图文写作，但设计本身是护城河
- 时间适配 4 — 周更 2 篇在 10h/周内
- 启动成本 5 — 0 元
- 收入天花板 3 — 起号后广告单价 500-3000/条，月入 2000-5000 合理

匹配理由：图文形式不露脸不直播；账号起来后能反哺 #1 模板售卖。
风险提示：起号慢是最大问题，前 100 篇可能零涨粉，需要和其他方向组合做。
冷启动第一步：定位垂直领域（如"给非设计师的 Figma 速查"），发 3 篇系列图文测试反馈。

## #5 表情包 / 贴纸数字产品（小红书 + 闲鱼虚拟商品）  22/30 分

评分明细：
- 技能匹配 4 — 板绘 OK，但要学商业化表情包规范
- 时间适配 5 — 单张 30 分钟，碎片时间友好
- 启动成本 5 — 0 元
- 收入天花板 2 — 客单价 1-10 元，跑量难，月入 500-2000 较现实
- 冷启动 4 — 一周可挂第一批
- 可持续性 2 — 内容消耗快，需持续产出新系列

匹配理由：碎片时间变现的兜底选项；可作为 #1、#2 的副产物。
风险提示：单品收入极低，单独做很难撑起一份副业，建议作为补充而不是主攻。
冷启动第一步：画一套 8 张的表情包（围绕一个 IP 形象），挂闲鱼虚拟商品，3.9 元/套。

## 总结建议

- **主推 #1**：技能直接变现、被动收入、风险低。
- **搭配 #2 或 #3 作为短期现金流**：#1 冷启动期（1-2 月）靠 #2/#3 维持动力。
- **#4 作为长线 IP**：和 #1 天然协同，但独立做风险高。
- **#5、#6 暂不单推**：#5 可作为副产物，#6 等新规落地后再评估。
