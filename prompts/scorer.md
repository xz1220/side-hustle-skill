# 副业评分标准

## 6 维度评分体系

每个维度 1-5 分，满分 30 分。打分必须结合用户画像——同一个副业方向对不同用户分数不同。

### 维度 1: 技能匹配度

评估用户现有技能和该方向所需技能的重合度。

| 分数 | 标准 |
|------|------|
| 5 | 用户已有的技能可以直接变现，几乎不需要额外学习 |
| 4 | 核心技能匹配，只需补充少量周边知识（< 1 周） |
| 3 | 有相关基础，需要 1-4 周学习新技能 |
| 2 | 有间接关联，需要 1-3 个月系统学习 |
| 1 | 几乎从零开始，需要 3 个月以上学习 |

### 维度 2: 时间适配度

评估用户可用时间是否能支撑该副业。

| 分数 | 标准 |
|------|------|
| 5 | 碎片时间就能做，完全适配用户时间安排 |
| 4 | 需要固定时间段但总量在用户预算内 |
| 3 | 刚好够用，但几乎没有余量 |
| 2 | 需要时间超出用户预算 20-50% |
| 1 | 需要时间远超用户预算，不可行 |

### 维度 3: 启动成本

评估启动所需资金与用户预算的匹配度。

| 分数 | 标准 |
|------|------|
| 5 | 零成本启动，只需要时间和已有设备 |
| 4 | 需要少量投入（< 200 元），在用户预算内 |
| 3 | 需要中等投入（200-1000 元），接近用户预算上限 |
| 2 | 需要较多投入（1000-5000 元），超出用户预算 |
| 1 | 需要大额投入（> 5000 元），远超用户预算 |

### 维度 4: 收入天花板

评估该方向的收入潜力。基于搜索到的真实数据，不是理论值。

| 分数 | 标准 |
|------|------|
| 5 | 中等水平月入 5000+，有明确的规模化路径 |
| 4 | 中等水平月入 2000-5000，有增长空间 |
| 3 | 中等水平月入 1000-2000，增长空间有限 |
| 2 | 中等水平月入 500-1000，天花板较低 |
| 1 | 中等水平月入 < 500，或收入极不稳定 |

### 维度 5: 冷启动难度

评估从零到第一笔收入的难度和时间。

| 分数 | 标准 |
|------|------|
| 5 | 一周内可以出第一单/第一笔收入 |
| 4 | 2-4 周内可以出第一笔收入 |
| 3 | 1-2 个月内可以出第一笔收入 |
| 2 | 3-6 个月才能看到收入 |
| 1 | 6 个月以上才可能有收入，前期全是投入 |

### 维度 6: 可持续性

评估该副业的长期价值。

| 分数 | 标准 |
|------|------|
| 5 | 可复利增长（内容/品牌/客户积累越做越值钱） |
| 4 | 收入稳定可持续，有一定积累效应 |
| 3 | 收入相对稳定，但没有积累效应（按件计费） |
| 2 | 收入不稳定，容易被替代（价格竞争激烈） |
| 1 | 短期窗口期，很快会被市场/技术淘汰 |

## 输出格式

按总分降序排列。每个方向：

```
┌──────────────────────────────────────────────┐
│ #{排名}  {方向名称}                {总分}/30 分 │
│                                              │
│ 评分明细：                                    │
│   技能匹配 {分} | 时间适配 {分} | 启动成本 {分} │
│   收入天花板 {分} | 冷启动 {分} | 可持续性 {分}  │
│                                              │
│ 匹配理由：{一句话}                             │
│ 风险提示：{一句话}                             │
│ 冷启动第一步：{具体行动}                       │
└──────────────────────────────────────────────┘
```

## 评分原则

1. **基于数据打分** — 每个分数都要有搜索数据或知识库支撑
2. **个性化打分** — 同一方向对不同用户分数不同（技能匹配、时间适配因人而异）
3. **保守而非乐观** — 宁可低估不要高估，用户信任比好看更重要
4. **说明理由** — 每个维度的分数都要有一句话解释为什么
